CÓMO NOS ORGANIZAMOS

🌿 Antes de comenzar nuestro proyecto, vamos a descubrir cómo trabajaremos para aprender juntos muchas cosas sobre el mundo de las plantas.

¡Así nos organizaremos!


1️⃣ Nos pondremos en equipos de 3 o 4 alumnos y alumnas. Cada grupo dispondrá de un ordenador y un pendrive para guardar sus investigaciones y actividades.

2️⃣ A cada equipo se le asignará un tema relacionado con las plantas para investigar. Además, podréis utilizar diferentes páginas web educativas con información fiable y adaptada a vuestra edad.

3️⃣ Para ayudaros durante la búsqueda, tendréis una ficha con preguntas y pistas que os servirán para encontrar la información más importante.

4️⃣ Después, compartiremos en clase todo lo que cada grupo haya descubierto y aprendido durante la investigación.

5️⃣ Finalmente, crearéis un trabajo digital en grupo utilizando Canva, donde recogeréis vuestros aprendizajes de una forma creativa y visual.
